# ChatBot
ChatBot using RNN and NLP with Simple local dataset
<hr>

## :dart:	 About: ##
  This is a simple chatbot using Recurrent neural network 'RNN' and Natural language processing 'NLP' , This is the initial version of this bot. The data on which the model was trained was written by the Team .
  
  :rotating_light: <b> wait for changes , this model will be developed soon</b> :rotating_light:
